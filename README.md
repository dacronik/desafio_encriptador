# desafio_encriptador_Alura

![portada](https://user-images.githubusercontent.com/96795408/218336526-d51a4768-7f4f-4c1b-8ebf-0ae8014453da.png)
Desafío encriptador de texto creado en HTML 5, CSS3 y JavaScript. El cual consiste en ingresar un texto cualquiera y poder encriptarlo, también tiene la funcionalidad
De poder copiar ese texto encriptado, mediante el botón copiar y luego poder desencriptar un texto. Este proyecto está desarollado dentro del marco del proyecto ONE de 
Oracle impartido en el bootcamp de desarrollo de aplicaciones Front-End de Alura para Latinoamérica 


![style-buttom](https://user-images.githubusercontent.com/96795408/218336951-888a9855-2bc0-43c9-91ee-822fe94e1822.png)
También se le quiso dar un toque personal, poniéndole una imagen de fondo al body desde el css y animaciones a los botones, dándoles un efecto de neón y 
dejando al proyecto responsivo para dispositivos moviles mediante media queries
